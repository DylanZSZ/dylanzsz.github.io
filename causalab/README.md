# causalab

Interactive write-up on **CausaLab**, a scalable environment for evaluating LLM agents as interactive causal discoverers — and what it reveals about how (and whether) they experiment like scientists.

Paper: *CausaLab: A Scalable Environment for Interactive Causal Discovery Toward AI Scientists.*
Code: https://github.com/DylanZSZ/CausaLab

## Local preview

```sh
python3 -m http.server 8000
# open http://localhost:8000/causalab/
```

## Stack

Plain HTML + CSS + a tiny `main.js`. No build step. Violet/aubergine theme.
Figures are rasterized from the paper's PDFs. Hosted on GitHub Pages from the `main` branch.
